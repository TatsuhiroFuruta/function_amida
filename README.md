# コース
### 1\. ジグザグコース<br>
直線 ($y=x$, $y=-x$) のみの通路が出現するコースです。

### 2\. 標準コース<br>
出現する通路において、ジグザグコースに三角関数、指数対数関数が追加されます。

### 3\. お楽しみコース<br>
出現する通路において、標準コースに逆三角関数、双曲線関数が追加されます。
